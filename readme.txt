File structure 
start with your node server path 
for my node server pahth is c:\nodejs
1. Put server.js in root nodejs 
  c:\nodejs\server.js

2. Create angularjs a folder under nodejs path and put shoppingList.html and all angular lib in it.
    C:\nodejs\angularjs\angular.js
    C:\nodejs\angularjs\shoppingList.html

3. Create controllers and filters folder under angularjs
   and put todoList.js and todoFilters.js 
   C:\nodejs\angularjs\controllers\todoList.js
   C:\nodejs\angularjs\filters\totoFilters.js   

4. Start node server as command below.
   >node server.js   

Lanuch your app with url --> localhost:5000/shoppingList.html
